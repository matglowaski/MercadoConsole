# Sistema de Mercado (Console Java)

Este é um projeto simples de terminal desenvolvido em Java que simula o funcionamento de um sistema de compras em um mercado. O objetivo principal foi colocar em prática conceitos fundamentais da linguagem, como manipulação de arrays, estruturas de repetição, condicionais e métodos estáticos.

## 🚀 Funcionalidades

- **Busca de Itens:** O usuário pode procurar produtos pelo nome.
- **Controle de Estoque:** O sistema verifica se há quantidade disponível antes de adicionar ao carrinho e atualiza o estoque após a venda.
- **Cálculo de Desconto:** Aplicação automática de descontos progressivos (5%, 10% ou 15%) dependendo do valor total da compra.
- **Relatório Simples:** Ao finalizar, o sistema mostra a média de preço dos produtos e permite filtrar itens abaixo de um determinado valor.

## 🛠️ Tecnologias Utilizadas

- **Linguagem:** Java
- **Entrada de Dados:** `java.util.Scanner`
